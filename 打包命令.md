# 打包命令

可执行程序（目录版）：

```powershell
.\.venv\Scripts\pyinstaller.exe --noconfirm --clean --windowed --name Baseband_Test_Data_Statistics_Tool --add-data "src/resources;src/resources" main.py
```

安装包版：

```powershell
& 'C:\Users\DJH\AppData\Local\Programs\Inno Setup 6\ISCC.exe' 'installer.iss'
```

在 cmd 里可直接这样执行（项目根目录下）：

```cmd
cd /d D:\RJHZ\Project_Code\Python\Baseband_Test_Data_Statistics_Tool

.\.venv\Scripts\pyinstaller.exe --noconfirm --clean --windowed --name Baseband_Test_Data_Statistics_Tool --add-data "src/resources;src/resources" main.py

"C:\Users\DJH\AppData\Local\Programs\Inno Setup 6\ISCC.exe" installer.iss
```
